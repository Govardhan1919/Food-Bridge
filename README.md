# Food Bridge 🍎🤝

Food Bridge is a comprehensive digital platform designed to bridge the gap between food donors and those in need. By streamlining the logistics of food donation, the platform connects donors, NGOs, volunteers, and riders to ensure that surplus food reaches its destination efficiently and safely.

## 🚀 Key Features

- **Role-Based Dashboards**: Tailored experiences for Donors, NGOs, Riders, Volunteers, Verifiers, and Admins.
- **Smart Logistics**: Real-time tracking and coordination for food pickup and delivery.
- **Secure Authentication**: Robust signup and login flows for all participant types.
- **Impact Tracking**: Monitor donations and their reach within the community.
- **User-Friendly Interface**: Modern, responsive design built for ease of use.

## 🛠️ Technology Stack

- **Frontend**: Vite, React, TypeScript, Tailwind CSS
- **UI Components**: shadcn-ui, Lucide React, Framer Motion
- **State Management**: TanStack Query (React Query)
- **Backend**: Node.js, Express (located in the `/server` directory)
- **Database**: PostgreSQL (with migration scripts included)

## 📦 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or bun

### Installation

1. **Clone the repository**:
   ```sh
   git clone https://github.com/Govardhan1919/Food-Bridge.git
   cd Food-Bridge
   ```

2. **Install dependencies**:
   ```sh
   npm install
   ```

3. **Configure Environment**:
   - Copy `.env.example` to `.env` and fill in the required details.

4. **Start the Development Server**:
   ```sh
   # Start frontend
   npm run dev

   # Start backend server (in a separate terminal)
   npm run server:dev
   ```
